Add preview button at fullscreen writing mode.

Latest Source: http://github.com/fukata/wp-fullscreen-preview-button/

== Installation ==

1. Download the plugin, unpack it and upload the 'wp-fullscreen-preview-button' folder to your wp-content/plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
